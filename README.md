# pastry-shop
Una semplice applicazione web CRUD

## Frontend

Ho usato React(Typescript) con Bootstrap

1) npm install
2) npm run start

## Backend

Ho usato .NET Core V8

1) dotnet run

### Attenzione
- Questa applicazione funziona solo su localhost, non è accesibile esternamente nemmeno nella stessa rete LAN.
- Non è responsive
